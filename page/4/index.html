TMNM计划 の Start

发表于 2019-09-04 更新于 2019-10-27

TMNM计划,即Take Math Note Myself计划,是博主Herself32新开设的一个项目.

之前很多关于数学的Blog,质量都比较低,也受到了不少读者的批评.

原因是参考的东西 太多,真正 自己总结出来的东西 没有几句话,真正能够拿出来让别人有学习价值的东西非常少.

一直在嚼别人剩下的东西,背别人的结论,背别人的推导.

背着背着,自己深刻理解的东西还是背之前的那么多,实质上 毫无进益.

所以TMNM计划将要改变这一倾颓的现状.

顾名思义,Take Math Note Myself,就是博主 完全不引用并尽量少地参考 他人的总结或文章,仅凭自己学过的知识和自身对于内容的总结写成博客.

希望TMNM计划,能够挽救博主现在对于数学学习的迷茫状态,和一直无法为读者提供高质量博客的现状.

千里之行,始于足下

ps: TMNM-1已经在咕了,因为第一次尝试TMNM,所以这玩意可能10月才咕出来
