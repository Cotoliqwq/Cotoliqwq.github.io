「是非成败转头空」- 我的第一场ACM

发表于 2019-07-28 更新于 2019-07-29

一场失败也成功的比赛

前言

来到ZR之后就一直期待这场ACM,虽然说之前了解过一些关于ACM的知识,但是自己亲身体验,还是第一次.

虽然打完之后觉得非常遗憾,但这也是一次非凡的经历

比赛前

大家处于欢乐的组队状态~

zs,RainAir,Logey组成的SD省队观光团,以及春神的个人单挑队被钦定最有可能夺冠的两个队.

没人要的我和莫名没队的cjc临时组成了双人团,不过在比赛前夜,毛老师为我们临时加上了JS的hrz.

比赛开始

cjc和hrz都没用过Linux…….

勉强先用我电脑开题…..,一人一道,cjc开$A$,我和hrz开$B,C$(后来才知道吃了顺序开题的亏)

大概在比赛7min的时候就有人切题了,但是我们队一点思路都没有.

15min+过去了,我们决定先看一眼那些题被A的最多…

于是发现了F,比较sb的multiset模板,hrz一时脑抽直接一发fhq,结果T了…..

乖乖写multiset,但是我虽然知道multiset和set的用法差不多,但是还是不太会用.

最后开了一下挂,大概会写了,cjc仍然在肝A,hrz看E…..

我在他俩坐在地上思考的时候敲了一发F,但是还是T…….(后来知道出题人卡cout)

cjc肝不动A了,于是和Hrz一起看E.

cjc: 这直接排个序然后二分不就完了吗?

在cjc把自己hack掉然后我俩又把cjc的hack hack掉之后,hrz写了一发,直接一遍过(Orz hrz)

这时候已经过去1h了.

开始自闭

A了一题之后cjc继续肝A,我和hrz看D.

hrz问我xor是不是有分配律,我想了想发现没有.

但是我发现其实一些$A_i + A_j$的xor是会对消的,但是没有继续往下想(如果想了之后就A了)

hrz看到了L:

hrz: cjc你看我是不是眼瞎了,这不是sb模拟吗

在大家验证hrz没有看错之后,他又一发过了L

于是我继续肝F,cjc肝不动A了之后来帮忙.

cjc: 这不是个蠢题吗

cjc爆出了$O(n)$正解,但是他又双叒叕没有清空答案(我把他hack了).

于是我们成为了罚时最多的队伍……

我继续看D,hrz开始看J.

cjc: 我发现A好蠢啊.

cjc鸽巢原理盲猜结论,然后我和hrz验证之后,cjc自告奋勇写了一发.

但是他把Yes和No写错了…….

然后我们队开始自闭…..,我看了J,说了一发思路,然后cjc和hrz说有道理,但是有个小细节一直处理不了….

hrz口胡了一发M,但是我们不会写Pollard-rho(虽然不需要)

继续看D,我给hrz说了对消这个发现:

hrz: 让开让开,我要切D

于是在4:00的时候我们把D终于切掉了.

然后去吃饭,我和cjc一起吃了碗兰州拉面,hrz点了华莱士.

最后半个小时了,hrz口胡了K,cjc口胡了I,我也把J的细节弄好了,我们也发现M完全不需要Pollard-rho……

但是半个小时四道题实在是太极限了,双开,甚至三开都不一定成功,何况单开….

于是我们颗粒无收….

赛后

hwy,xry他们队力克春神拿到rank1,SD省队选手观光团连rk前三都没有进入…..

虽然这和我们仅仅A了5个题的队相差甚远,但是我们从这场比赛总结了教训:

  1. 决不能顺序开题
  2. 不能照着一个题死磕
  3. 有思路就要往下想
  4. 注意细节

花絮

赛后第二天Scape老师整了一个在vjudge上的个人图论练习赛.

我凭借着做过两题的优势,直接拿到了三个题的一血,最后solved9个题拿到rk1

hrzsolved 7拿到rk4,不过值得一提的是,春神又跨班比赛了,而且又双叒叕创造了两个小时6题的纪录(本菜鸡八个小时才九个题啊)

THE END
